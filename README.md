## Project Description

This project is a responsive personal portfolio website developed using HTML5 and CSS3.

The portfolio showcases my education, technical skills, internship experience, projects, certifications, and contact information in a modern and professional interface.

The website is fully responsive and works on desktop, tablet, and mobile devices.

---

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts

---

## Features

- Responsive Navigation Bar
- Hero Section
- About Me
- Skills with Progress Bars
- Experience Timeline
- Projects Section
- Certifications Section
- Contact Form
- Footer
- CSS Hover Effects
- CSS Animations
- Mobile Responsive Design

## How to Run the Project

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Open `index.html`.
5. Right-click and select **Open with Live Server**.
